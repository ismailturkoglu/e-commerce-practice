# Smile-Trade Project

A simple **e-commerce frontend UI project** built with HTML, CSS, and Bootstrap.

## 🔗 Live Demo


## About the Project

Smile-Trade is a static frontend template that includes product listing, product details, and shopping cart pages.

## Pages

- **Home (index.html)**  
  Product cards, category menu, and carousel slider.

- **Product Details (details.html)**  
  Product images, description, and similar products section.

- **Shopping Cart (shopping-cart.html)**  
  Cart items and order summary layout.

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- Font Awesome
- fsLightbox

## Features

- Responsive layout
- Bootstrap grid system
- Product card design
- Image gallery with lightbox
- Shopping cart UI

## Project Structure

```
index.html
details.html
shopping-cart.html
css/style.css
img/
```

## Purpose

Built for practicing frontend development and understanding e-commerce UI structure.
